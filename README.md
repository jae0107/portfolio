# My Portfolio Website

This is my responsive portfolio website developed in React.

## Install dependencies

    npm install

## Usage

    npm i node-sass
    npm i @material-ui/core
    npm i @material-ui/icons
    npm i ityped
    npm install emailjs-com --save
    npm i react-modal-video
